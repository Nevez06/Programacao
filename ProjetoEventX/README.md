# EventX - Plataforma de Gestão de Eventos

Sistema completo para criação, gestão e interação com eventos.

## Funcionalidades

- **Organizadores**: Login, criação de eventos, gestão de convites, administração financeira
- **Fornecedores**: Login e cadastro
- **Convidados**: Login via evento, chat em tempo real
- **IA**: Assistente virtual integrado
- **Design**: Totalmente responsivo com tema EventX

## Configuração

Configure o arquivo `.env` com suas credenciais e execute as migrations do Entity Framework.
